# Till Game

A Python checkout/till game for Compute Club.

## Run on Raspberry Pi

1. Create the virtual environment:

python3 -m venv .venv

2. Activate it:

source .venv/bin/activate

3. Install requirements:

pip install -r requirements.txt

4. Run the game:

python main.py
